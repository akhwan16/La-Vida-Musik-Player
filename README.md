# Fitur

1. Tambahkan lagu dengan file
2. Tambahkan lagu dengan folder
3. Lagu sebelumnya
4. Lagu berikutnya
5. Jeda
6. Pengatur volume
7. Senyap
8. Ulang
9. Putar
10. Hapus lagu
11. Cari
