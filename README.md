# Features

1. Select songs
2. Previous song
3. Next song
4. Pause / Unpause song
5. Music time slider
6. Volume slider
7. Mute / Unmute button
8. User-friendly interface

# Supported music files

1. \*.mp3
2. \*.wav
3. \*.flac

# Requirements

```
pygame
mutagen
```

For installing requirements: `pip install -r requirements.txt`

# Run

```
python main.py
```
